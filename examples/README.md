# Vox Examples

示例程序放在 `examples/`，可复用包放在 `pkgs/`。

推荐示例：

- `examples/hello`：最小包 + 测试
- `examples/c_demo`：测试、FFI、构建产物
- `examples/wasm_call`：Node/Web 调用 wasm
- `examples/llmdemo`：通过 `pkgs/llm` 调 OpenAI
- `examples/vox_claw`：agent/tool 流程，调用 OpenAI

相关包：

- `pkgs/openai`
- `pkgs/genai`
- `pkgs/llm`
